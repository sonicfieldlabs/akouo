# /one-sound-many-ears

## Purpose

`/one-sound-many-ears` is the comparative flagship command. It runs one sonic object through multiple listening modes and compares what each reveals, hides, risks, and cannot know.

The goal is not total interpretation. The goal is accountable plurality.

## When To Use

Use `/one-sound-many-ears` for:

- comparing listening modes
- teaching or demonstrating akoúō
- research method development
- sound design exploration
- sonic artwork analysis
- prompts or recordings with several possible readings
- identifying contradictions between technical, perceptual, musical/aesthetic, affective, archival, ecological, political, and speculative ears

## Execution Order

1. Run `signal-inspection-listening`.
2. Run `acoulogical-object-listening`.
3. Run `embodied-affective-listening`.
4. Run `transductive-media-listening`.
5. Run `forensic-archival-listening`.
6. Run `ecological-posthuman-listening`.
7. Run `critical-political-listening`.
8. Run `musical-aesthetic-listening`.
9. Run `symbolic-fictional-listening`.
10. Compare mode outputs for contradictions, productive tensions, overreach risks, and methodological value.
11. Produce the most responsible reading and the most interesting next mode.

## Expected Output

Use `schemas/comparative-listening-output.schema.json`.

The output must include:

- `sound_object`
- `mode_comparison`
- `shared_claim_summary`
- `contradictions`
- `productive_tensions`
- `most_responsible_reading`
- `most_interesting_next_mode`

## Guardrails

- Preserve differences between modes.
- Do not average the ears into one generic answer.
- If a mode has little evidence, say so instead of forcing content.
- Keep speculative mode clearly marked.
- Keep forensic mode conservative.
- Keep signal inspection from claiming cultural meaning.

## Must Not Do

- Do not claim total knowledge because many modes were used.
- Do not hide contradictions.
- Do not let the most poetic reading dominate the responsible reading.
- Do not let the technical reading erase embodied, mediated, ecological, political, or symbolic dimensions.
- Do not ignore `undetermined`.

## Recommended Follow-Up

Recommend the mode with the highest methodological value for the user's next action. For research, often recommend `/study` or `/reference`. For evidence, recommend `/forensic`. For production, recommend `/tech` or `/full-ear`.
